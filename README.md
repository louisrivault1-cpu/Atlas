# ATLAS — BTS GTLA Révisions

Application web de révisions pour le BTS GTLA (Gestion des Transports et Logistique Associée).

## Fonctionnalités

- Fiches de révision par matière
- Quiz interactifs avec correction
- Exercices à choix multiples
- Interface responsive et moderne
- Application single-page (fichier HTML unique)

## Utilisation

Ouvrir `index.html` dans un navigateur web, ou accéder via GitHub Pages.
